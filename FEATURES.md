## 기술 참고

### 클라이언트 사이드 라이브러리
- QR 코드: `qrcode` 또는 `qrcode.vue`
- Diff: `diff` 또는 `jsdiff`
- 마크다운: `marked` 또는 `markdown-it`
- 코드 하이라이팅: `highlight.js` 또는 `prism`
- 날짜: `dayjs` (가벼움)

### 브라우저 API 활용
- 클립보드: `navigator.clipboard`
- 파일 다운로드: `URL.createObjectURL` + `<a download>`
- 카메라 (QR 스캔): `navigator.mediaDevices.getUserMedia`
- 로컬 저장: `localStorage`
